# The poetry of Anne Steele

This contains OCR'd text from books of Anne Steele's poetry, all found at Internet Archive.

The 3 volumes of her works (but this is published after her death):

https://archive.org/details/ubjectsc01stee V1 `steel_vol_1.md`
https://archive.org/details/subjectsc02stee V2 `steel_vol_2.md`
https://archive.org/details/poectsc03stee Miscellaneous `steel_vol_3.md`

Internet Archive has two copies of Sheppard’s collection:

https://archive.org/details/hymnspsalmspoems00stee 1868 edition `sheppard-1868.md`
https://archive.org/details/hymnspsalmspoems00stee_1 1882 edition (not yet OCR'd, and I probably won't)

The Sheppard collection was easier to OCR; it doesn't use a long s, or use the old convention of having
the first word of the next page appear at the bottom of the current page. We'll see if we can get
everything cleaned up.
